# Nico-Netflix Comment Overlay

Netflixの動画再生画面上に、ニコニコ生放送のコメントをリアルタイムで流す（弾幕表示する）ためのGoogle Chrome拡張機能です。
スポーツ中継（WBCなど）やアニメの同時視聴など、別プラットフォームのコメントを見ながらNetflixの映像を楽しみたい場合に最適です。

## ✨ 主な機能 (Features)

* **ニコ生コメントのリアルタイム弾幕表示**: HTML5 Canvasを使用して、Netflixのプレイヤー上にニコ生のコメントを軽量かつ滑らかにオーバーレイ表示します。
* **サイドバーUI**: Netflixの画面右側に専用のサイドバーを展開し、コメントの履歴一覧を確認できます。
* **ニコ生へのコメント投稿**: サイドバーの入力欄からコメントを送信すると、裏で接続しているニコニコ生放送へ実際のコメントとして投稿されます。
* **ディレイ（遅延）調整機能**: 配信と動画のタイミングが合わない場合、拡張機能のポップアップからコメントの表示タイミング（秒数）を微調整して同期させることができます。
* **シミュレーションモード**: 実際の番組IDの代わりに `test` と入力することで、オフラインで弾幕のテスト描画を行えます。

## 🚀 インストール方法 (Installation)

本拡張機能はChromeウェブストアで公開されていないため、デベロッパーモードから手動で読み込む必要があります。

1. このリポジトリをZIPでダウンロードするか、`git clone` します。
   ```bash
   git clone [https://github.com/zyuuuukak1n/nico-netflix_comment_overlay.git](https://github.com/zyuuuukak1n/nico-netflix_comment_overlay.git)
   ```
2. Google Chromeのアドレスバーに `chrome://extensions/` と入力し、拡張機能の管理画面を開きます。
3. 画面右上の **「デベロッパーモード」** のスイッチをオンにします。
4. 左上の **「パッケージ化されていない拡張機能を読み込む」** をクリックし、ダウンロードしたリポジトリのフォルダ（`manifest.json` が含まれているディレクトリ）を選択します。

## 📖 使い方 (Usage)

1. Netflixを開き、視聴したい動画の再生画面（URLが `https://www.netflix.com/watch/` から始まるページ）を開きます。
2. ブラウザ右上の拡張機能ツールバーから、本拡張機能のアイコンをクリックしてポップアップを開きます。
3. **「NicoNico Live ID」** の欄に、同期したいニコニコ生放送の番組ID（例: `lv123456789`）を入力します。
   *※テスト動作を確認したい場合は `test` と入力してください。*
4. **「Start Overlay on Netflix」** ボタンをクリックします。
5. バックグラウンドでニコ生のタブがミュート状態で自動的に開かれ、Netflixの画面上にサイドバーが展開されます。コメントが取得されると画面右から左へ弾幕が流れます。
6. 映像とコメントのタイミングがずれている場合は、拡張機能のポップアップを開き、**「Comment Delay Offset (seconds)」** のスライダーを動かして表示タイミングを調整してください。
7. 終了する場合は、ポップアップの **「Stop Overlay」** をクリックするか、Netflixのタブを閉じてください。

## 📂 ファイル構成 (Project Structure)

* `manifest.json` - Chrome拡張機能の設定ファイル (Manifest V3)
* `background.js` - ニコ生のタブ管理、コメントのルーティング、テストモードのシミュレーション処理
* `/popup` - 拡張機能アイコンをクリックした際の設定画面UI (ID入力、開始/停止、遅延設定)
* `/content` - ページに直接注入されるスクリプト群
  * `content.js` - Netflix側の初期化、サイドバーとCanvasのDOM挿入
  * `danmaku.js` - HTML5 Canvasを利用した弾幕描画エンジンと遅延処理
  * `nico_content.js` - ニコ生側に注入され、DOMの変化を監視してコメントを抽出・送信するスクリプト
  * `ui.js` - Netflix上のサイドバーの開閉、コメント履歴表示、投稿処理
  * `styles.css` - サイドバーやNetflixプレイヤー領域のスタイル調整

## ⚠️ 注意事項と免責事項 (Disclaimer)

* 本ツールは非公式の拡張機能です。Netflixおよびニコニコ動画の公式サービスとは一切関係ありません。
* 各サイトのUIやDOM構造が変更された場合、コメントの取得や描画が正常に機能しなくなる可能性があります。
* バックグラウンドでニコニコ生放送のページを開く仕様上、ニコニコ動画のアカウントにログインしている必要があります（ログインしていないとコメントが取得できない場合があります）。
